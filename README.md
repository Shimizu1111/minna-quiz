# みんなのクイズ

小学校（高学年向け）の4択クイズアプリです。HTML/CSS/JavaScriptだけで動く軽いつくりで、ネット上に無料で公開できます。

## ファイルの役割

| ファイル | 中身 | 編集する？ |
|---|---|---|
| `index.html` | 画面の骨組み | 基本そのまま |
| `style.css` | 見た目・色 | 色を変えたいとき |
| `app.js` | クイズの動き | 基本そのまま |
| `questions.js` | **問題データ** | ★ここを編集して問題を増やします |

## 問題の増やし方（いちばん大事）

`questions.js` を開いて、下のかたまりをコピーして増やすだけです。

```js
{
  category: "理科",                          // 教科の名前（同じ名前でまとまります）
  q: "水が氷になることを何という？",           // 問題文
  choices: ["蒸発", "凝固", "融解", "沸騰"],  // 選択肢を4つ
  answer: 1,                                 // 正解が上から何番目か（0,1,2,3）
},
```

- `answer` は **いちばん上が 0**、その次が 1 … と数えます。上の例は「凝固」が正解です。
- 本番では選択肢の並び順は自動でシャッフルされます。
- `category` を新しい名前にすると、スタート画面に教科ボタンが自動で増えます。

## 手元で動かして確認する

このフォルダで以下のどちらか：

- VS Code の「Live Server」拡張で `index.html` を開く
- または `npx serve` を実行してブラウザで `http://localhost:3000` を開く

## ネットに公開する（GitHub Pages・無料）

1. GitHub でリポジトリを作り、このフォルダ一式をアップロード（push）する
2. リポジトリの **Settings → Pages** を開く
3. 「Build and deployment」の Source を **Deploy from a branch** にし、Branch を **main / (root)** にして Save
4. 数分待つと `https://（ユーザー名）.github.io/（リポジトリ名）/` で公開されます

このURLを知っていれば、誰でもブラウザから遊べます。
